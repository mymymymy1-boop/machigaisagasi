# 「まちがいさがし」 — 5才むけ 小学校受験対策 PWA

## 概要

5才のお子さま向けの「まちがいさがし」PWA (Progressive Web App) です。
小学校受験対策の観察力・集中力トレーニングを目的に、家庭内で繰り返し遊べるよう設計されています。

- **モードA (まちがいさがし)**: 2枚の絵を見比べて、ちがう箇所を5つ見つける
- **モードB (なかまはずれさがし)**: 4枚の絵のなかから、1枚だけちがう絵を見つける
- 各モード **100問** ずつ、合計 **200問** 収録
- **40シーン** (動物・乗り物・食べ物・公園・学校・季節・自然・家庭) を題材
- **完全オフライン動作** (Service Worker による precache、初回オンライン時にキャッシュ)
- インストール後はネット接続不要、サーバー通信ゼロ
- 広告・トラッキング・外部送信なし

## クイックスタート (3つのデプロイ方法)

### a) Cloudflare Pages (ドラッグ&ドロップ)

1. https://dash.cloudflare.com にログイン
2. 「Workers & Pages」→「Create application」→「Pages」→「Upload assets」
3. `C:/dev/machigaisagasi` フォルダごとドラッグ&ドロップ
4. デプロイ完了後、配布された `*.pages.dev` のURLをiPhone/iPadのSafariで開く

### b) Netlify Drop (ドラッグ&ドロップ)

1. https://app.netlify.com/drop を開く
2. `C:/dev/machigaisagasi` フォルダごとドラッグ&ドロップ
3. 配布された `*.netlify.app` のURLをSafariで開く

### c) ローカル (家庭内LAN)

```powershell
cd C:/dev/machigaisagasi
python -m http.server 8000
```

ブラウザで http://localhost:8000 を開く。
同一Wi-FiのiPhone/iPadからは `http://<PCのIPアドレス>:8000` でアクセス可。

## インストール手順 (iPhone)

1. Safari で配布URLを開く
2. 画面下の **共有ボタン** (□に↑のアイコン) をタップ
3. メニューから **「ホーム画面に追加」** を選択
4. 名前を確認して右上の **「追加」** をタップ
5. ホーム画面に「まちがいさがし」アイコンが追加される

## インストール手順 (iPad)

1. Safari で配布URLを開く
2. 画面右上の **共有ボタン** をタップ
3. メニューから **「ホーム画面に追加」** を選択
4. 右上の **「追加」** をタップ
5. ホーム画面にアイコンが追加される

> 一度ホーム画面から起動すれば、以後は **オフライン動作** します。
> (Service Worker が全アセット・全シーン・全問題データを precache するため)

## +30問 自動追加 (Claude API)

問題セットを増やしたい場合、Anthropic Claude API を使って自動生成できます。

```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-..."
node tools/generate-more.js --mode A --count 30
```

オプション:
- `--mode A` or `--mode B` — どちらのモードに追加するか
- `--count 30` — 追加する問題数

生成された問題は `data/modeA-problems.json` または `data/modeB-problems.json` に追記されます。

## 注意

- **家庭内利用前提** です。商用配布や公開サイトでの提供は想定していません。
- 配布URLが第三者に知られないよう、Cloudflare Pages / Netlify のアクセス制限機能の利用を推奨します。
